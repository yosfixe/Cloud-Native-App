Hotel Booking API

Cloud-Native Microservices — Youssef Kazti

All requests go through the API Gateway at http://localhost:8080

Room Service

GET
/rooms
List all rooms
GET
/rooms/:id
Get a room by ID
POST
/rooms
Create a room — body: number, type, price, capacity
PUT
/rooms/:id
Update a room
PATCH
/rooms/:id/availability
Toggle availability — body: available (boolean)
DELETE
/rooms/:id
Delete a room

Guest Service

GET
/guests
List all guests
GET
/guests/:id
Get a guest by ID
POST
/guests
Register a guest — body: name, email, phone
PUT
/guests/:id
Update a guest
DELETE
/guests/:id
Delete a guest

Reservation Service

GET
/reservations
List all reservations
GET
/reservations/:id
Get a reservation by ID
POST
/reservations
Create a reservation — body: roomId, guestId, checkIn, checkOut
DELETE
/reservations/:id
Cancel a reservation

Gateway Health

GET
/gateway/health
Check API Gateway status
